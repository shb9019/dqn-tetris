# DQN Tetris
Python Version: 3.6+

## Setup
`git clone https://github.com/shb9019/dqn-tetris.git`

`virtualenv dqn-tetris -p python3`

`cd dqn-tetris`

`pip install torch==0.3.1 numpy`

## Instructions to Run

Training the model - `python dqn.py`

Running trained model - `python run.py model_best.pth.tar`
